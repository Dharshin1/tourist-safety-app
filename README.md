# Tourist Safety App

An AI-powered mobile application designed to enhance tourist safety using real-time location tracking and emergency response features.

## Features
- Live Google Maps integration
- SOS emergency alert
- Location sharing
- Nearby safe place detection

## Tech Stack
- Flutter
- Google Maps API
- Geolocator
- Firebase (planned)
